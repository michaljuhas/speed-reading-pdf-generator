# Speed-reading PDF generator

## Folders

* `Input`
  * `EN` - here are the source files
* `Output` - an empty folder which will include the generated PDF files
* `Scripts` - includes `pdf-generator.py`

## Run

`cd Scripts`

`python pdf-generator.py`
